# API ドキュメントジェネレーター

以下の手順で API ドキュメントを生成してください：

1. `/src/api/` 内のすべてのファイルをスキャン
2. 関数シグネチャと JSDoc コメントを抽出
3. エンドポイント/モジュール別に整理
4. 例付きの Markdown を作成
5. リクエスト/レスポンスのスキーマを含める
6. エラードキュメントを追加

出力形式：
- `/docs/api.md` に Markdown ファイル
- すべてのエンドポイントに curl の例を含める
- TypeScript の型を追加
