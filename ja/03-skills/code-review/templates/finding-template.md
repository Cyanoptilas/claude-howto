### コード例

#### 現在（問題あり）

```typescript
// N+1 クエリ問題を示す
const users = fetchUsers();
users.forEach(user => {
  const posts = fetchUserPosts(user.id); // ユーザーごとにクエリ！
  renderUserPosts(posts);
});
```

#### 修正案

```typescript
// JOIN クエリで最適化
const usersWithPosts = fetchUsersWithPosts();
usersWithPosts.forEach(({ user, posts }) => {
  renderUserPosts(posts);
});
```
