# コードレビューチェックリスト

## セキュリティチェックリスト
- [ ] ハードコードされたクレデンシャルやシークレットがない
- [ ] すべてのユーザー入力に入力バリデーションがある
- [ ] SQL インジェクション対策（パラメータ化クエリ）がある
- [ ] 状態変更操作に CSRF 保護がある
- [ ] 適切なエスケープによる XSS 対策がある
- [ ] 保護されたエンドポイントに認証チェックがある
- [ ] リソースに認可チェックがある
- [ ] セキュアなパスワードハッシュ（bcrypt、argon2）を使用している
- [ ] ログに機密データがない
- [ ] HTTPS が強制されている

## パフォーマンスチェックリスト
- [ ] N+1 クエリがない
- [ ] インデックスを適切に使用している
- [ ] 効果的なキャッシュが実装されている
- [ ] メインスレッドでブロッキング操作がない
- [ ] async/await が正しく使用されている
- [ ] 大きなデータセットにページネーションがある
- [ ] データベース接続がプールされている
- [ ] 正規表現が最適化されている
- [ ] 不必要なオブジェクト生成がない
- [ ] メモリリークが防止されている

## 品質チェックリスト
- [ ] 関数が50行未満
- [ ] 変数名が明確
- [ ] 重複コードがない
- [ ] 適切なエラーハンドリング
- [ ] コメントが WHAT ではなく WHY を説明している
- [ ] 本番環境に console.log がない
- [ ] 型チェック（TypeScript/JSDoc）がある
- [ ] SOLID 原則に従っている
- [ ] デザインパターンが正しく適用されている
- [ ] 自己文書化コードになっている

## テストチェックリスト
- [ ] ユニットテストが書かれている
- [ ] エッジケースがカバーされている
- [ ] エラーシナリオがテストされている
- [ ] 統合テストがある
- [ ] カバレッジが80%以上
- [ ] フレーキーなテストがない
- [ ] 外部依存関係がモックされている
- [ ] テスト名が明確
