# コードレビュースキル

このスキルは以下に焦点を当てた包括的なコードレビュー機能を提供します：

1. **セキュリティ分析**
   - 認証・認可の問題
   - データ漏洩リスク
   - インジェクション脆弱性
   - 暗号化の弱点
   - 機密データのログ記録

2. **パフォーマンスレビュー**
   - アルゴリズムの効率性（Big O 分析）
   - メモリの最適化
   - データベースクエリの最適化
   - キャッシュの機会
   - 並行性の問題

3. **コード品質**
   - SOLID 原則
   - デザインパターン
   - 命名規則
   - ドキュメント
   - テストカバレッジ

4. **保守性**
   - コードの読みやすさ
   - 関数のサイズ（50行未満が望ましい）
   - 循環的複雑度
   - 依存関係の管理
   - 型の安全性

## レビューテンプレート

レビューするコードの各部分について：

### サマリー
- 全体的な品質評価（1〜5）
- 主な発見数
- 推奨優先領域

### クリティカルな問題（ある場合）
- **問題**: 明確な説明
- **場所**: ファイルと行番号
- **影響**: なぜ重要か
- **重大度**: Critical/High/Medium
- **修正**: コード例

### カテゴリ別の発見

#### セキュリティ（問題が見つかった場合）
例と共にセキュリティ脆弱性をリスト

#### パフォーマンス（問題が見つかった場合）
複雑度分析と共にパフォーマンス問題をリスト

#### 品質（問題が見つかった場合）
リファクタリング提案と共にコード品質問題をリスト

#### 保守性（問題が見つかった場合）
改善案と共に保守性の問題をリスト
